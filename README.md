# NormalizerProject

This project normalizes job titles using Java and Maven.

## Prerequisites

- JDK 8 or higher
- Maven 3.6.0 or higher

## How to Run

1. **Clone the repository:**

   ```sh
   git clone https://github.com/your-username/NormalizerProject.git
   cd NormalizerProject
   ```

2. **Compile the project:**

   ```sh
   mvn clean compile
   ```

3. **Run the main class:**

   ```sh
   mvn exec:java -Dexec.mainClass=com.example.SimpleJobTitleNormalizer
   ```

4. **Run the tests:**

   ```sh
   mvn test
   ```
```

Replace `https://github.com/your-username/NormalizerProject.git` with the actual URL of your repository if needed.
